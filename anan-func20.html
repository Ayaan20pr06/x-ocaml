Jane Street OCaml Challenge: Higher-Order Functions

Functions in OCaml are first-class values. You can pass them to other functions, store them in variables, and return them from functions.

map_option

map_option applies a function to an option if it is Some. If the value is None, it propagates that.

let map_option f opt = match opt with | None -> None | Some i -> Some (f i) let double i = 2 * i let () = assert ([%compare.equal: int option] (map_option double None) None) let () = assert ([%compare.equal: int option] (map_option double (Some 2)) (Some 4)) (* Using an anonymous function *) let () = assert ([%compare.equal: int option] (map_option (fun i -> 2 * i) (Some 2)) (Some 4))
